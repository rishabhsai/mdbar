Plain files

Your notes stay readable without the app.

Daily notes live in a folder you choose. Regular notes are just markdown files too. No account, no hidden database, no export ceremony.

Hybrid editing

Markdown stays markdown, but it doesn’t look raw.

Headings, lists, and checkboxes render in place while you type, so the editor feels close to the final document instead of making you bounce between edit and preview.

Built for muscle memory

Open it with one shortcut and keep moving.

mdbar lives in the menu bar, respects dark mode, supports tab indentation, and gives the small preferences that make daily use feel personal.

Open source

Built in public, with an MIT license and a commit history worth reading.
